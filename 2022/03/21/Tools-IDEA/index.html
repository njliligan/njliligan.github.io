HEAD头文件

UTF-8

  1. Setting -> Editor -> FileEncodings
    Project Encoding -> UTF-8
    Default encoding for properties files -> UTF-8
    请不要勾选with bom选项
  2. 打开intelliidea安装路径，IntelliJIDEA15.0.6\bin找到idea.exe.vmoption文件，
    打开，在最后一行添加：”-Dfile.encoding=utf-8
  3. 通常是tomcat，设置idea server编码。在菜单栏找到”run->editconfigration” 找到”server”选 项卡 设置 vm option为 -Dfile.encoding=utf-8。
  4. 菜单栏 -> HELP->Edit Custom VM OPtions中加 -Dfile.encoding=utf-8
